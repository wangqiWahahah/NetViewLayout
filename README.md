# NetViewLayout

## 开发原因  
开发中我们可能会遇到这种需求，在节日或者特定时间时我们需要显示一些特定的布局和简单的内容，并增加简单的点击，跳转功能。<br>
这样我们就需要以服务器为准，根据服务器下发的坐标，调转去向，来进行布局。<br>
